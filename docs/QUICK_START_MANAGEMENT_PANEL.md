### 📊 التبويبات الأربعة:

#### 1️⃣ **📦 الأذونات:**
```
Columns:
├─ # (الرقم)
├─ رقم الأذن (#0001)
├─ المورد (اسم الشركة)
├─ الوزن الفعلي (كيلو)
├─ تاريخ الاستلام
├─ الحالة (✅ مسجلة / 🏭 في الإنتاج)
├─ الفاتورة (✅ / ❌)
└─ الإجراءات (👁️ / ✏️ / 🗑️)

Buttons:
• 👁️ عرض   → عرض جميع التفاصيل
• ✏️ تعديل  → تحديث الوزن والتاريخ والملاحظات
• 🗑️ حذف   → حذف الأذن والسجلات المرتبطة
```

#### 2️⃣ **📄 الفواتير:**
```
Columns:
├─ # (الرقم)
├─ رقم الفاتورة (#INV-001)
├─ المورد (اسم الشركة)
├─ تاريخ الفاتورة
├─ الحالة (⏳ / ⚙️ / ✅ / ❌)
├─ عدد الأذونات (2)
└─ الإجراءات (👁️ / ✏️ / 🗑️)

Buttons:
• 👁️ عرض   → عرض تفاصيل الفاتورة
• ✏️ تعديل  → تحديث البيانات والحالة
• 🗑️ حذف   → حذف الفاتورة والفصل عن الأذونات
```

#### 3️⃣ **⚖️ سجلات التسوية:**
```
Columns:
├─ # (الرقم)
├─ الأذن (#0001)
├─ الفاتورة (#INV-001)
├─ الوزن الفعلي (كيلو)
├─ وزن الفاتورة (كيلو)
├─ الفرق (كيلو) ← يحمر إذا كبير
├─ النسبة (%) ← يعرض الفرق المئوي
├─ الحالة (✓ / ⚠️ / 📝 / ❌)
└─ الإجراءات (👁️ / ✏️ / 🗑️)

Buttons:
• 👁️ عرض   → عرض التفاصيل الكاملة
• ✏️ تعديل  → تحديث الأوزان والحالة
• 🗑️ حذف   → حذف السجل وإرجاع الأذن
```

#### 4️⃣ **🔄 الحركات المسجلة:**
```
Columns:
├─ # (الرقم)
├─ رقم الحركة (#MOV-001)
├─ النوع (⚙️ تعديل / ⚖️ تسوية)
├─ الكمية (كيلو)
├─ المصدر (reconciliation)
├─ التاريخ والوقت
├─ الحالة (✅ مكتمل)
└─ الإجراءات (👁️ / 🗑️)

Buttons:
• 👁️ عرض   → عرض تفاصيل الحركة
• 🗑️ حذف   → حذف الحركة من السجل
```

### ⚡ الإجراءات السريعة:

```
مثال عملي: تصحيح فاتورة بها خطأ

الخطوات:
1. اضغط على تبويب "الفواتير"
2. ابحث عن رقم الفاتورة: INV-001
3. اضغط على ✏️ تعديل
4. غيّر الرقم من INV-001 إلى INV-0001
5. اضغط 💾 حفظ التعديلات
6. ✅ تم التحديث!

مثال 2: حذف أذن مسجلة بالخطأ

الخطوات:
1. اضغط على تبويب "الأذونات"
2. ابحث عن الأذن #0001
3. اضغط على 🗑️ حذف
4. أكد الحذف في النافذة
5. ✅ تم الحذف (مع جميع السجلات المرتبطة)
```

### 🎓 نصائح للاستخدام الفعّال:

```
✅ استخدم البحث للعثور على البيانات بسرعة
✅ افتح التفاصيل قبل التعديل للتأكد من البيانات
✅ تحقق من الفروقات قبل حذف التسويات
✅ احفظ نسخة من التقارير قبل عمليات التعديل الكبيرة
✅ استخدم الملاحظات لتوثيق التعديلات
```

---

### ⚠️ تحذيرات مهمة:

```
❌ لا تحذف فاتورة بدون التأكد من عدم الحاجة لها
❌ لا تعدّل الأوزان الفعلية إلا إذا كنت متأكداً
❌ احفظ صورة من البيانات الأصلية قبل التعديلات الكبيرة
❌ لا تحذف سجلات التسوية إلا في حالات الخطأ
```

---

### 🔐 السلامة:

```
✅ جميع العمليات محمية بـ CSRF Token
```
