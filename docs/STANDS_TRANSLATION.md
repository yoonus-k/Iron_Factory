# نظام الترجمة لصفحات الاستاندات

## نظرة عامة
تم إنشاء ملف ترجمة شامل لجميع صفحات إدارة الاستاندات (Stands) في نمطية Manufacturing.

## الملفات المُنشأة

### ملف الترجمة الرئيسي
- **المسار**: `resources/lang/ar/stands.php`
- **الوصف**: يحتوي على جميع نصوص الترجمة العربية لصفحات الاستاندات

## الصفحات المُحدثة

تم تطبيق الترجمة على الصفحات التالية:

### 1. صفحة إنشاء الاستاند (Create)
- **المسار**: `Modules/Manufacturing/resources/views/stands/create.blade.php`
- **النصوص المُترجمة**:
  - عنوان الصفحة
  - رؤوس الأقسام
  - تسميات الحقول
  - نصوص المساعدة
  - نصوص الأزرار
  - رسائل التنبيه

### 2. صفحة تعديل الاستاند (Edit)
- **المسار**: `Modules/Manufacturing/resources/views/stands/edit.blade.php`
- **النصوص المُترجمة**:
  - عنوان الصفحة
  - رؤوس الأقسام
  - تسميات الحقول
  - خيارات الحالة
  - نصوص الأزرار
  - سكريبت تحديث حالة الحالة النشطة

### 3. قائمة الاستاندات (Index)
- **المسار**: `Modules/Manufacturing/resources/views/stands/index.blade.php`
- **النصوص المُترجمة**:
  - عنوان الصفحة
  - رؤوس الجدول
  - حقول البحث والفلترة
  - خيارات الحالة
  - أزرار الإجراءات

  - رسائل التأكيد على الحذف

### 4. تفاصيل الاستاند (Show)
- **المسار**: `Modules/Manufacturing/resources/views/stands/show.blade.php`
- **النصوص المُترجمة**:
  - عنوان الصفحة
  - رؤوس البطاقات الرئيسية
  - تسميات المعلومات
  - نصوص الأزرار
  - رسائل التأكيد على الحذف

### 5. تاريخ الاستخدام (Usage History)
- **المسار**: `Modules/Manufacturing/resources/views/stands/usage-history.blade.php`
- **النصوص المُترجمة**:
  - عنوان الصفحة
  - تسميات الفلاتر
  - خيارات الحالة
  - الإحصائيات
  - رؤوس الجدول

## كيفية الاستخدام

### في نموذج Blade:

```blade
<!-- عرض النص المُترجم -->
<h1>{{ __('stands.form.stand_number') }}</h1>

<!-- مع المتغيرات -->
<p>{{ __('stands.message.no_stands') }}</p>

<!-- في Attributes -->
<input placeholder="{{ __('stands.placeholder.stand_number') }}">
```

### في JavaScript:

```blade
<script>
    const message = '{{ __('stands.alert.confirm_delete') }}';
    if (confirm(message)) {
        // تنفيذ الإجراء
    }
</script>
```

## الخيارات المدعومة حالياً

### حالات الاستاند:
- `unused` - غير مستخدم
- `stage1` - المرحلة الأولى
- `stage2` - المرحلة الثانية
- `stage3` - المرحلة الثالثة
- `stage4` - المرحلة الرابعة
- `completed` - مكتمل
- `in_use` - قيد الاستخدام
- `returned` - مُرجع

### حالات النشاط:
- `active` - نشط
- `inactive` - غير نشط

## ملاحظات مهمة

1. **استخدام الدوال الترجمة**: تأكد من استخدام دالة `__()` أو `trans()` في جميع النصوص.
2. **الترتيب العربي**: جميع النصوص مُعدة للعربية من اليمين إلى اليسار.
3. **التوسع المستقبلي**: يمكن إضافة لغات إضافية بنفس البنية في مجلدات جديدة تحت `resources/lang/`.
4. **التحديثات**: عند إضافة نصوص جديدة، قم بإضافتها إلى ملف الترجمة بنفس نمط التنظيم.

## مثال على الإضافة:

إذا كنت تريد إضافة نص ترجمة جديد:

```php
// في resources/lang/ar/stands.php
'new_key' => 'النص الجديد',

// في البلايد:
{{ __('stands.new_key') }}
```
